# Security Policy

## Supported Versions

Currently, the following versions are being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report any security vulnerabilities by opening an issue or contacting the repository owner directly. Do not disclose sensitive security issues publicly until they have been patched.

For Vertex AI backend issues, ensure that your `API_KEY` or `GOOGLE_APPLICATION_CREDENTIALS` are not exposed in the codebase or public environments.
