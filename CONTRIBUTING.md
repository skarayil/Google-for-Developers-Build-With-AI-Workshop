# Contributing to Akıllı Ev Otomasyonu

First off, thank you for considering contributing to Akıllı Ev Otomasyonu! It's people like you that make open source such a great community.

## Where do I go from here?

If you've noticed a bug or have a feature request, please [open an issue](#) on GitHub.

## Fork & create a branch

If this is something you think you can fix, then [fork Akıllı Ev Otomasyonu](#) and create a branch with a descriptive name.

## Implement your fix or feature

At this point, you're ready to make your changes. Feel free to ask for help; everyone is a beginner at first.

## Make a Pull Request

At this point, you should switch back to your master branch and make sure it's up to date with Akıllı Ev Otomasyonu's master branch.
Then push your branch to your fork and create a Pull Request against the `main` branch.
